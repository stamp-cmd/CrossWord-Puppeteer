# CrossWord-Puppeteer

A Puppeteer code to brute-force crossword for website: <https://www.boatloadpuzzles.com/playcrossword>

## Installation

install node.js and puppeteer.
```sh
npm install
```

## Running

run code by node.js
```sh
node Cross.js
```

## Dependencies

Puppeteer automatically install latest chromium.
